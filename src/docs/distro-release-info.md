## Current Release

- **Name:** LinuxHub Preview
- **Version:** 0.1.0
- **Status:** Preview / Developer Build

This build is intended for testing the LinuxHub experience: package manager UI, configuration panels, and system monitoring. It is not yet recommended for production systems.

## Highlights

- Modern desktop UI built with React, Tailwind CSS, and Tauri.
- Integrated package view with install queue and live progress.
- System configuration panels for:
  - System info and hardware overview
  - Kernel management
  - Updates and storage
  - Devices, network, and locale

## Known Limitations

- Some configuration actions may be simulated or partially implemented.
- Hardware support and driver helpers are still evolving.
- UX and visuals are under active development and may change between builds.
